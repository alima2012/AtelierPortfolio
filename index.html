Description

Cette application permettait de gérer la gestion du personnel du Groupement d'incendie et de secours N2 de Thiès. L'objectif était de leur fournir une application permettant de gérer l'ensemble du personnel, les mutations: à chaque fois que l'on affecte un agent d'un GIS à un autre alors, on devrait faire de sorte que l'application envoie un rappel pour signaler qu'un agent doit être muté. Après mutation, l'agent muté ne devrait plus figuré sur la liste du personnel, toutes ses données doivent être envoyées dans le GIS auquel il a été muté etc.Les langages et technologies utilisées était: PHP, HTML/CSS, Bootstrap, PowerAMC Nous devrions fournir les pages suivantes:

  • Fournir une page de connexion
  • Une page de gestion du personnel
  • Une page gérant les mutations
  • Une page permettant d'intégrer les rapports de sortie etc...
